基础类型

user
用户基本信息
名称 类型 说明
id int 用户id
name string 用户名
default_head bool 是否使用默认头像
package
图包基本信息
名称 类型 说明
id int 图包id
title string 图包标题
file
文件信息
名称 类型 说明
hash string 文件hash
width int 宽度
height int 高度
image
图片信息
名称 类型 说明
id int 图片id
file file 图片对应文件
imageDetail
图片信息
名称 类型 说明
id int 图片id
package package 所属图包
user user 创建用户
file file 图片对应文件
description string 图片描述
praised bool 是否被赞过
comments coment[] 评论
time string 创建时间
packageDetail
图包详细信息
名称 类型 说明
id int 图包id
title string 图包标题
thumbs Image[] 缩略图,最多四张,第一张为封面图
imageCount int 图包内图片数量
user user 图包创建人
followerCount int 关注人数
description string 图包描述
praised bool （图包非自己创建时）是否赞过
followed bool （图包非自己创建时）是否关注
userDetail
用户详细信息
名称 类型 说明
id int 用户id
name string 用户名称
default_head bool 是否使用默认头像
follows_count int 关注数
followers_count int 粉丝数
packages_count int 图包数
images_count int 图片数
praise_count int 赞过的数量
followed bool （图包非自己创建时）是否关注
comment
评论
名称 类型 说明
id int 评论id
user user 此评论的用户
text string 评论的内容
time string 评论的时间
mentions mention[] 评论@的人
mention
评论提及的人
名称 类型 说明
user_id int 提及的人的id
pos int 开始的位置
len int 长度

页面数据

all
名称 类型 说明
user user 当前已经登录的用户,若未登录,user.id为0,其余两个参数忽略
images imageDetail[] 图片列表
following
名称 类型 说明
user user 当前已经登录的用户,若未登录,user.id为0,其余两个参数忽略
images imageDetail[] 图片列表
package
名称 类型 说明
user user 当前已经登录的用户,若未登录,user.id为0,其余两个参数忽略
package packageDetail 图包详情
sub1 string 子页1,可能的取值"follower"
datas Array 数组,不同情况有不同取值
image
名称 类型 说明
user user 当前已经登录的用户,若未登录,user.id为0,其余两个参数忽略
image imageDetail 图片信息
user
名称 类型 说明
user user 当前已经登录的用户,若未登录,user.id为0,其余两个参数忽略
page_user userDetail 图片信息
sub1 string 子页1,可能的取值("follower"."following","image","praise")
sub2 string 子页2,可能的取值(following{"package"},praise{"package"})
datas Array 附带的数组,根据不同的情况有不同类型
